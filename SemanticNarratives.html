Semantic Narratives

Semantic Narratives allow you to embed and visualise underlying data in narratives allowing your narratives to convey information that narratives by themselves find difficult and to relate local micro description to the larger macro picture.

ResearchSpace allows data to be embedded in Web Narratives so that data is visualised and carried by a traditional document. This document acts as a standalone database in its own right.
